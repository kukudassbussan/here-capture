# Project Blueprint - URL Capture

## Overview
A modern web application designed to capture full-page screenshots of websites via a URL input. Built with standard web technologies and a focus on high-end aesthetics and user experience.

## Project Details
### Design & Features
- **Theme:** "Midnight Aura" - Dark background with deep shadows and vibrant glow effects.
- **Typography:** Inter, System-ui. Expressive sans-serif fonts with emphasized hierarchy.
- **Components:**
    - **Smart Input Bar:** Validates URLs in real-time.
    - **Capture Engine:** Utilizes Microlink API for high-resolution snapshots. Optimized for community sites (DC Inside, Femco) by handling dynamic content and hiding overlays.
    - **Gallery View:** Displays the captured screenshot in a polished frame.
    - **Persistence:** Remembers recent captures for quick access using localStorage.

### Technical Stack
- **HTML5:** Semantic structure.
- **CSS:** Baseline 2024 features (Container queries, `:has()`, Logical properties, Cascade Layers).
- **JavaScript:** ES Modules, Async/Await.
- **API:** Microlink.
