# URL Capture | Full-Page Screenshot Tool

A modern, high-performance web application that allows users to capture full-page screenshots of any website. 

## Features
- **Full-Page Capture:** Generates high-resolution images of entire web pages.
- **Midnight Aura Design:** A visually stunning dark theme with modern CSS features.
- **Local History:** Remembers your recent captures for easy access.
- **Downloadable:** Save screenshots directly to your device.
- **Powered by Microlink:** Fast and reliable screenshot engine.

## Built With
- Vanilla HTML/CSS/JavaScript
- CSS Cascade Layers & Container Queries
- Microlink API

## Getting Started
1. Open `index.html` in your browser.
2. Enter a URL (e.g., `https://github.com`).
3. Click **Capture** and wait a few seconds.
4. Preview the screenshot and click **Download** to save it.
